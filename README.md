# infant-cry-detection-cnn
## Overview
This project detects infant cry sounds using machine learning techniques and audio signal processing.

## Methodology
- Audio preprocessing
- MFCC feature extraction
- Dataset balancing
- Classification using SVM and KNN
- Prediction of cry category

## Technologies Used
- MATLAB
- Machine Learning
- Signal Processing
- MFCC Features

## Files
- extract_features.m → Feature extraction
- train_model.m → Model training
- test_model.m → Testing and prediction

## How to Run
1. Run train_model.m to train the model
2. Run test_model.m and select audio file
